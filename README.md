# Know Your Calories
A mod for the game "Project Zomboid"

## Description:
### Shows your Character's Calories on the Info Tab

```
How it works:
After obtaining cooking level 5 or the nutritionist trait, your player's calories will now show up in the Info Tab. 

(Both the level needed and the trait interaction can be changed or disabled in the sandbox options)
```
